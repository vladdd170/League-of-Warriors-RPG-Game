# League of Warriors – RPG Game

A text-based RPG adventure game implemented in Java, built around core OOP principles.

## Features
- Three playable character classes: Warrior (Fire immunity, Strength), Mage (Ice immunity, Charisma), Rogue (Earth immunity, Dexterity)
- Turn-based combat system with normal attacks and typed spells (Fire, Ice, Earth)
- Procedurally generated grid map (up to 10x10) with enemies, sanctuaries, and a portal
- Character progression: XP gain, level-up, attribute scaling
- Account system with JSON parsing (credentials, characters, game history)
- Custom exceptions: InvalidCommandException, ImpossibleMoveException
- 50% chance mechanics for damage doubling/halving and enemy immunities

## Tech
Java · OOP · Abstract classes · Interfaces · JSON parsing · Custom exceptions
